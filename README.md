# todo-list

## For Vue setup
----------------------------------------
## step 1 : instal node_modules
    npm install

## step 2 : compile and run
    npm run serve

## step 3 : for build
    npm run build


## For Json Server
---------------------------------------
## step 1 : Install json server
    npm install -g json-server

## step 2 : Go to server Folder and run below command
    json-server --watch todolist.json
